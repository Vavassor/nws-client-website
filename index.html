NWS Client

NWS Client is a JavaScript library for the National Weather Service API.

Get started

Why NWS Client?

Free

The NWS API is a public service of the United States Government and there are no fees to use it.

TypeScript

NWS Client supports TypeScript and is written in it. It helps to maintain types, which are complicated for NWS API data.

Platforms

Web Browsers, Node.js, and React Native, and Electron are all supported platforms.
